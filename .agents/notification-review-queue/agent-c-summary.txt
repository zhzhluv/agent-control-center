===========================================
주요 기능
===========================================

✓ 검수 대기 에이전트 목록 표시
✓ 상태별 색상 구분
  - pending: 강한 초록 (애니메이션)
  - acknowledged/copied: 연한 초록
  - dismissed: 회색
✓ 시간순 정렬 (오래된 것이 위)
✓ 개수 배지 (pending/처리됨/숨김)

✓ 반응형 레이아웃 (모바일/태블릿)

□ 항목 클릭: Inspector에 자동 선택
□ 상태 변경: UI 실시간 업데이트
